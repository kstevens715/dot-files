# In home directory:
```
$ cd dot-files
$ chmod +x ./install
$ ./install

$ ln -s dot-files/gitconfig .gitconfig # Only if you're me.
```
In vim `:PluginInstall`

If you are on a MAC you will need to brew install ack, and change ack-grep to ack

## Prerequisites
```
# Install ack for searching:
sudo pacman -S ack
```
