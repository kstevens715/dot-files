# In home directory:
```
$ cd dot-files
$ chmod +x ./install
$ ./install

$ ln -s dot-files/gitconfig .gitconfig # Only if you're me.
```
In vim `:PluginInstall`

## Prerequisites
```
# Install the_silver_searcher (ag) for searching:
sudo pacman -S ag

# Install and run ctags alternative
gem install ripper-tags
ripper-tags -R
```
